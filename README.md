# Petunjuk Penggunaan

Untuk menjalankan server.js, Anda harus menginstall terlebih dahulu Node JS pada komputer Anda. Setelah itu, lakukan langkah-langkah sebagai berikut :

1. **Download** file `server.js` yang ada di halaman ini.
2. **Buka CMD** pada laptop dengan cara:
   - Tekan `Win + R`
   - Lalu ketik `CMD` dan tekan **Enter**.
3. **Jalankan file** dengan mengetik pada cmd: 
   `node alamatfile`
4. **Contoh** menjalankan di cmd:
   ```bash
   node C:\download\server.js
